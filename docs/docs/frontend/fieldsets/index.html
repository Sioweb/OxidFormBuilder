Fieldsets

Fieldset

Fieldsets sind Container, die eine Gruppe von Eingabefeldern definieren. Sobald Sie einige Felder einem Formular hinzugefügt haben, können Sie diese nach einem Neuladen auf der rechten Seite einer Gruppe zuweisen.

Sortieren

Klicken und halten Sie mit der linken Maustaste den kleinen Balken unterhalb einer Gruppe und ziehen Sie die Gruppe an die gewünschte Position.

Warnung: Einige Elemente wurden nicht gefunden

Diese Meldung erscheint, falls Eingabefelder gelöscht wurden, welche diesem Formular noch zugeordnet waren. Dieser Felder können nicht widerhergestellt noch analysiert werden. Es kann lediglich darüber Informiert werden, wie viele Felder nicht mehr vorhanden sind.
