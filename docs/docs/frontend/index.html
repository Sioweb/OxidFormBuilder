Formulare im Frontend nutzen

Oxid bietet von Haus aus, wenig Möglichkeiten der Kommunikation zwischen Nutzer und Shopbetreiber. Formulare müssen pro Anwendugnsfall durch Module nachgerüstet werden.

Mit dem FormBuilder, können Sie nun ein Formular erzeugen, einstellen und beliebig in Templates und CMS-Seiten einfügen.

Auf den nachfolgenden Seiten erfahren Sie, wie Sie Eingabefelder und Formulare konfigurieren müssen, um Ihren Besuchern Formulare bieten zu können.
