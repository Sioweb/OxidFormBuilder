SSH

Mit SSH auf einen Server verbinden

Auf dieser Seite erfahren Sie rudementär, wie sie per SSH auf einen Server zugreifen können. Es werden keine Umfangreichen Kenntnisse vorausgesetzt, allerdings sollten Sie sich mit den gängigen Konsolenbefehlen vertraut machen: ls -la, cd, mv, cp, rm. Bitte beachten Sie, dass wir keinen Support für Verbindungen zu Hostern leisten, bitte fragen Sie bei Bedarf Ihren Hoster, oder ziehen Sie das Forum zu rate.

Damit Sie sich mit Ihrem Server verbinden können, benötigen Sie einen SSH-Zugang zu Ihrem Server. Diesen erfahren Sie im Kundenbereich Ihres Servers, oder von Ihrem Serveranbieter/Hoster.

Hinweis: Bitte beachten Sie, dass es nicht empfehlenswert ist, direkt an einer Live-Version Ihres Shops zu arbeiten.

Windows

Unter Windows gibt es zwei empfohlene Möglichkeiten.

  • Putty
  • Git for Windows

Putty

Putty ist eine SSH fähige Clientanwendung. Sie können hier Ihren Daten für Ihren Server hinterlegen und sich per Knopfdruck und eingabe des SSH-Passwortes verbinden

Git for Windows

Dieses Tool liefert das Programm Git Bash bei der Installation. Über diese Konsole können Sie sich per ssh username@ihr-shop.de auf Ihren Server verbinden. Nach absenden des Befehls ist die Eingabe des Passwortes nötig.

Die Konsole wird Sie beim ersten Verbinden auffordern den SSH Fingerprint zu speichern. Akzeptieren Sie diese aufforderung.

Linux/Max OSx

Über die Konsole können Sie sich per ssh username@ihr-shop.de auf Ihren Server verbinden. Nach absenden des Befehls ist die Eingabe des Passwortes nötig.

SSH ohne Passwort

Sie können es sich leicht machen, und ein Schlüsselpaar erzeugen. Dieses erlaubt Ihnen sich per SSH ohne Passworteingabe zu verbinden. Plattformen wie Gitlab bieten umfangreiche Informationen zur Erstellung von Schlüsselpaaren.
