Forms

Formulare konfigurieren

Formulare bestehen aus zwei Bereichen. Links wird das Formular selbst definiert und wie es reagieren soll, wenn es vollständig und erfolgreich abgeschickt wird.

Auf der rechten Seite, kann ein Stück weit das Aussehen des Formulares definiert werden. Sobald Sie einige Eingabefelder hinzugefügt haben, können Sie hier die Eingabefelder via Drag&Drop sortieren und verschiedenen Gruppen zuordnen.

Formular als E-Mail senden

Wollen Sie das Formular per E-Mail senden, haken Sie die Option Formular senden an. Es werden nun weitere Eingabefelder erscheinen, die Sie konfigurieren müssen, damit die E-Mail gesendet werden kann.

E-Mail Template

Eine E-Mail wird in der Regel als HTML und als Plaintext abgesendet, um möglichst alle E-Mail-Programme zu erreichen.

Das Plaintext-Format kann direkt im Formular selbst definiert werden. Dazu können alle Eingabefelder die Sie auf der Rechten Seite hinterlegt haben, genutzt werden. Wenn Sie als Beispiel das Eingabefeld mit dem Feldnamen street angelegt haben, können Sie in das Plaintext-Feld folgendes hinterlegen:

street: [{ $STREET }]

Bitte verwenden Sie hier keine Umlaute oder Sonderzeichen.
