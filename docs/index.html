Introduction

Einleitung

Das ist die offizielle Dokumentation des Formbuilders, für Anwender und Entwickler.

Als Anwender, werden Sie erfahren wie Sie den FormBuilder nutzen können, um individuelle Formulare im Frontend ausgeben zu können. Entwicklern bietet diese Dokumentation eine umfassende Beschreibung, wie FormBuilder für Backendformualre von Module und Core im Backend verwendet werden kann.
